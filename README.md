# BeatStream Music Player

Premium desktop-grade local music player platform built with HTML5, CSS3, Bootstrap 5, and Vanilla JS modules.
Runs entirely client-side without any backend database servers.

## Core Engines
- **Web Audio API Engine**: 5-band Equalizer, Stereo Panning balance, and real-time procedurally generated synthesizer tracks (offline fallback).
- **IndexedDB Database Layer**: Local persistence for uploaded MP3/WAV tracks, storing media as binary blobs.
- **Lyrics Sync Engine**: LRC/LRCX synchronization with line auto-scrolling and word-by-word karaoke text color gradients.
- **Audio Visualizers**: High-fidelity HTML5 Canvas rendering for Oscilloscope Waves, Frequency Spectrum Bars, and reactive circular beat visualizers.
